# Papiloscopista PC-PR — PWA

App de estudos instalável para celular, funciona **offline**.

## Como publicar no GitHub Pages (3 passos)

### 1. Crie um repositório no GitHub
- Acesse https://github.com/new
- Nome sugerido: `papiloscopista-pcpr`
- Deixe **Public**
- Clique em **Create repository**

### 2. Suba os arquivos

```bash
git init
git add .
git commit -m "primeiro commit"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/papiloscopista-pcpr.git
git push -u origin main
```

### 3. Ative o GitHub Pages
- No repositório, vá em **Settings → Pages**
- Em "Source", selecione **main** e pasta **/ (root)**
- Clique em **Save**

Após 1–2 minutos seu app estará em:
```
https://SEU_USUARIO.github.io/papiloscopista-pcpr/
```

## Como instalar no celular

**Android (Chrome):**
1. Abra o link no Chrome
2. Toque nos 3 pontinhos (⋮)
3. "Adicionar à tela inicial"

**iPhone (Safari):**
1. Abra o link no Safari
2. Toque no botão de compartilhar (□↑)
3. "Adicionar à Tela de Início"

Pronto — aparece como app na tela inicial e funciona offline!
